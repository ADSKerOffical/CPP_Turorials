C++ является одним из самых популярных и быстрых языков программирования. Он очень быстрый в вычислениях среди тех же Python и Lua потому что C++ компилированный язык.
На нём написано много всяких программ и он может напрямую взаимодействовать с железом. Вот основные черты которые нужно знать:
1) Чтобы написать функцию нужно сделать так:

int main() {
  return 0;
}

В C++ есть много типов, но вот некоторые из них: {
  string – строка
  bool – булевое значение (true/false)
  float/doble/int – целое число
  void – отсутствие типа
}

! функция main ВСЕГДА должна быть в коде и обозначена int, иначе скрипт просто не сработает
! функцию main НИКОГДА нельзя вызывать иначе это вызовет ошибку
? для каждого завершения инструкции нужно использовать точку с запятой, иначе компилятор не знает где заканчивается сама инструкция
? return делает так чтобы функция возвращало что-то

Чтобы использовать библиотеки нужно использовать #include и типо <LibraryName>. Для примера iostream позволяет выводить текст в консоль. Пример:

#include <iostream>

void run() {
    std::cout << "Text" << std::endl;
}

int main() {
    run();
    return 0;
}

Если ты хочешь чтобы можно было добавлять строки, то ты можешь использовать так:

#include <iostream>

std::string text2 = "Text";
void run(std::string text) {
    std::cout << text << " Example" << std::endl;
}

int main() {
    run(text2); // Text Example
    return 0;
}

? Для того чтобы сделать комментарии можно использовать два слеша: //
? std нужен для предотвращения конфликта имён
! В C++ типизация аргументов очень строгая и нужно каждый раз обозначать тип данных
! Если в функции хоть один аргумент не получен при действии то выведет ошибку

Ты можешь создавать новые переменные, но обозначать тип данных обаятельно:

#include <iostream>

std::string text2 = "Example";
void run(std::string text) {
    std::cout << text << std::endl;
}

int main() {
    run("Text " + text2); // Text example
    return 0;
}

Для функции можно использовать аргументы. Для создания нескольких аргументов их нужно разделять запятой:

#include <iostream>

int run(std::string text) {
    std::cout << text << std::endl;
}

int main() {
    run("Text");
    return 0;
}

Для переноса строки ты можешь использовать новый endl (end line)

#include <iostream>

int run(std::string text, std::string text2) {
    std::cout << text << std::endl;
    std::cout << text2 << std::endl;
}

int main() {
    run("Text", "Text2"); // Text \n Text2
    return 0;
}
